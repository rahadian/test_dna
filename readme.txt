Saya menggunakan docker compose dan python framwork fastapi untuk mengerjakan test. Pastikan docker sudah terinstall.
1. Deploy aplikasi pada direktori yang diinginkan.
2. Masuk pada direktori project yaitu test_dna/app.
3. Masukkan perintah "docker compose build" atau "docker-compose build" dan tunggu hingga proses build selesai.
4. Masukkan perintah "docker compose up -d" atau "docker-compose up -d" untuk menjalankan project.
5. Untuk akses aplikasi melalui alamat localhost:8000

 Terimakasih.
